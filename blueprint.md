# Grade Calculator

## Overview

This project is a web-based grade calculator that allows users to input their courses, grades, and credits to calculate their Grade Point Average (GPA).

## Features

*   **Course Input:** Users can add multiple courses with their corresponding grades and credit hours.
*   **GPA Calculation:** The application automatically calculates the GPA based on the provided information.
*   **Dynamic UI:** The user interface is built with modern web technologies to provide a seamless and interactive experience.
